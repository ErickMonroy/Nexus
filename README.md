**Calzando a México**: Sistema de gestión para zapatería.

## Descripción
Es una plataforma web diseñada para la zapatería Calzando a México.
Su objetivo es gestionar de forma eficiente las operaciones de personal, inventario y ventas  de la empresa, ofreciendo una interfaz moderna y funcional.

---

## Requisitos del Sistema

**Software necesario:**
- PHP 8.4.4
- MySQL Workbench 8.0
- Navegador moderno (Chrome)
- Servidor local (IIS sobre Windows recomendado)

---
## Tecnologías Utilizadas  

- PHP: Backend, conexión a base de datos.  
- JavaScript (JS): Automatización de procesos para el análisis de las métricas de desempeño
- HTML: Estructura y contenido del sistema.  
- CSS: Diseño visual de la interfaz. 
- MySQL: Gestión de la base de datos.  
- IIS (Internet Information Services): Servidor web sobre Windows.

---

## Tablas de la Base de Datos

- **Personal**: Información del personal interno y externo de la empresa (organigrama).
- **Productos**: Datos de los productos que se manejan usandu SKU.
- **Inventario y ventas**: Registro de inventario para los produtos dentro de las tiendas.

----
## Secciones Principales

- Inicio
- Persona
- Productos
- Inventario
- Conexión a la base de datos: El sistema establece una conexión centralizada.
